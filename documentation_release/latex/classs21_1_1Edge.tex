\doxysubsection{Detailed Description}
Represents an edge in a figure, connecting two vertices. 

This class models a directed edge in a graph structure. Each edge has a starting vertex ({\ttfamily begin\+\_\+}) and an ending vertex ({\ttfamily end\+\_\+}).

\begin{DoxyNote}{Note}
In test mode ({\ttfamily \#ifdef TEST}), the member variables are public for testing purposes. 
\end{DoxyNote}
