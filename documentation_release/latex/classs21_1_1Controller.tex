\doxysubsection{Detailed Description}
Provides an interface for interacting with Model in MVC pattern. 

The \mbox{\hyperlink{classs21_1_1Controller}{Controller}} class is responsible for managing the interaction between the GUI and the underlying 3D model. It provides methods for scaling, moving, rotating, and reading the model, as well as resetting its position and retrieving its data.
