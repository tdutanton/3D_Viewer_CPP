\doxysubsection{Detailed Description}
Represents a point in 3D space with homogeneous coordinates. 

The {\ttfamily \mbox{\hyperlink{structs21_1_1Point}{Point}}} struct defines a point with coordinates (x, y, z, w), where w is the homogeneous coordinate (default is 1.\+0). It provides methods for transformation, comparison, and printing, as well as getters and setters for the coordinates.
