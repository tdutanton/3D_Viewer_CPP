\doxysubsection{Detailed Description}
Represents a vertex in 3D space, derived from \mbox{\hyperlink{classs21_1_1SceneObject}{Scene\+Object}}. 

The {\ttfamily \mbox{\hyperlink{classs21_1_1Vertex}{Vertex}}} class encapsulates a point in 3D space and provides functionality for transforming the vertex and comparing it with other vertices. \begin{DoxyNote}{Note}
In test mode ({\ttfamily \#ifdef TEST}), the member variables are public for testing purposes. 
\end{DoxyNote}
