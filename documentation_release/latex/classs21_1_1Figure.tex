\doxysubsection{Detailed Description}
Represents a figure in a 3D scene, derived from \mbox{\hyperlink{classs21_1_1SceneObject}{Scene\+Object}}. 

The {\ttfamily \mbox{\hyperlink{classs21_1_1Figure}{Figure}}} class encapsulates a 3D figure, including its vertices, faces, edges, and name. It provides functionality for transforming the figure, printing its properties, and managing its geometry. The class is a friend of {\ttfamily \mbox{\hyperlink{classs21_1_1FileReader}{File\+Reader}}} and {\ttfamily \mbox{\hyperlink{classs21_1_1Facade}{Facade}}} to allow controlled access to its private members.

\begin{DoxyNote}{Note}
In test mode ({\ttfamily \#ifdef TEST}), the member variables are public for testing purposes. 
\end{DoxyNote}
