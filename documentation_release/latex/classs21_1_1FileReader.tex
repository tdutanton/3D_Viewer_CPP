\doxysubsection{Detailed Description}
A class for reading figure data from a file. 

This class provides functionality for reading figure data from a file, processing the data, and normalizing vertex coordinates. \begin{DoxyNote}{Note}
In test mode ({\ttfamily \#ifdef TEST}), the member variables are public for testing purposes. 
\end{DoxyNote}
