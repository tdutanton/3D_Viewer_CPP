\doxysubsection{Detailed Description}
Represents interface for interacting with \mbox{\hyperlink{classs21_1_1Controller}{Controller}} class. 

The \mbox{\hyperlink{classs21_1_1Facade}{Facade}} class is central point of access to model part of MVC pattern. Contains \mbox{\hyperlink{classs21_1_1FileReader}{File\+Reader}}, \mbox{\hyperlink{classs21_1_1Figure}{Figure}} and \mbox{\hyperlink{classs21_1_1SceneDrawerBase}{Scene\+Drawer\+Base}}. It provides API for operations like loading, moving, rotating, and scaling a 3D figure.
