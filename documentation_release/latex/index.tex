{\bfseries{3DViewer v2.\+0}} is an application for visualizing 3D wireframe models, developed in C++ following object-\/oriented programming principles. This project is an improved version of the previous application (3DViewer v1.\+0) and includes new features, support for large datasets, and flexible interface settings.\hypertarget{index_autotoc_md3}{}\doxysection{Features}\label{index_autotoc_md3}

\begin{DoxyItemize}
\item Load 3D models in {\ttfamily .obj} format (vertices and face lists only).
\item Perform the following affine transformations on the model\+:
\begin{DoxyItemize}
\item Translate along the X, Y, Z axes.
\item Rotate around the X, Y, Z axes.
\item Scale the model.
\end{DoxyItemize}
\item Graphical User Interface (GUI) that includes\+:
\begin{DoxyItemize}
\item A button to select the model file and a field to display its name.
\item A visualization area for the wireframe model.
\item Control elements for translating, rotating, and scaling the model.
\item Information about the uploaded model (file name, number of vertices and faces).
\end{DoxyItemize}
\item Support for models with vertex counts ranging from to 1,000,000 and more without noticeable delays (interface freeze does not exceed 0.\+5 seconds).
\item Implementation based on the MVC architectural pattern.
\item Use of three different design patterns (Facade, Builder, Singleton).
\end{DoxyItemize}\hypertarget{index_autotoc_md4}{}\doxysection{Additional Features}\label{index_autotoc_md4}
\hypertarget{index_autotoc_md5}{}\doxysubsection{Settings}\label{index_autotoc_md5}

\begin{DoxyItemize}
\item Choose the type of projection (parallel or central).
\item Configure the type (solid, dashed), color, and thickness of edges, as well as the shape, color, and size of vertices.
\item Change the background color.
\item Save user settings between program runs.
\end{DoxyItemize}\hypertarget{index_autotoc_md6}{}\doxysubsection{Recording}\label{index_autotoc_md6}
