\doxysubsection{Detailed Description}
Abstract base class representing a scene object. 

The {\ttfamily \mbox{\hyperlink{classs21_1_1SceneObject}{Scene\+Object}}} class serves as a base class for all objects in a 3D scene. It defines an interface for transforming objects using a transformation matrix. Classes derived from {\ttfamily \mbox{\hyperlink{classs21_1_1SceneObject}{Scene\+Object}}} must implement the {\ttfamily transform} method.
